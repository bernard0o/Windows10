O que é o Windows 10?

O Windows 10 é uma das versões do sistema operacional feito pela Microsoft. Este é responsável pela interface do computador.

Essa versão veio após o Windows 8 (que foi considerado um fracasso). Com essa versão, a Microsoft trouxe novas funcionalidades e agradou grande parte do público.

Estão disponíveis duas versões do Windows 10:

  • Windows 10 Home
  • Windows 10 Pro

Também há outras versões, mas estas não estão disponíveis diretamente ao público. Como por exemplo:

  • Windows 10 Mobile
  • Windows 10 Enterprise
  • Windows 10 Education
